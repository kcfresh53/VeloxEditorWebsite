Changelog

Release notes

Every version of Velox Editor, newest first. Follow along as the timeline, Script FX, and export pipeline keep getting faster.

v1.4.0LatestJuly 8, 2026

Scriptable clips leave beta with a faster sandbox and a preset library.

  • NewScript FX presets: insert bar graphs, waveforms, lower-thirds, and pulses from a single menu.
  • NewSandboxed worker rewrite renders Script FX previews roughly 40% faster on long clips.
  • Improvedfx helper library gains fx.lerp and a smoother fx.smooth easing curve.
  • ImprovedScript FX editor now shows clip time and frame index while scrubbing.
  • FixedFixed a rare freeze when a Script FX clip threw inside a tight loop.
v1.3.0June 3, 2026

Motion tracking gets faster and subtitles gain confidence controls.

  • NewSSD motion tracker can now track against a composited background.
  • NewAuto subtitle generator adds a minimum-confidence slider and target-track picker.
  • ImprovedKeyframe editor bezier handles snap to common easing presets.
  • FixedCorrected drift when baking tracked motion into position keyframes.
v1.2.0May 12, 2026

Hardware acceleration expands across every major platform.

  • NewGPU decode and encode auto-detect for VideoToolbox, NVENC, VA-API, QSV, and AMF.
  • NewAutomatic fallback ladder drops to CPU encoding when a GPU path fails.
  • ImprovedExport dialog surfaces the active encoder so you can confirm the GPU path.
  • FixedResolved a green-tint issue when transcoding full-range yuvj420p sources.
v1.1.0April 18, 2026

Color, chroma key, and transitions arrive.

  • NewSeven-way color correction: temperature, tint, brightness, contrast, saturation, highlights, shadows.
  • NewChroma key with similarity, smoothness, and spill suppression.
  • NewFade, slide, zoom, and wipe transitions with directional control.
  • ImprovedVisual cubic-bezier editor for fade-in and fade-out curves.
v1.0.0March 2, 2026

The first stable release of Velox Editor.

  • NewMulti-track timeline for video, audio, image, and text with drag-and-drop import.
  • NewTransform tools with per-property keyframe animation.
  • Newffprobe-based media import with automatic browser-safe transcoding.
  • NewExport to MP4, WebM, and MOV with configurable resolution, frame rate, and bitrate.
